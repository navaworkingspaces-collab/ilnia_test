# PHP React Test

## Requisitos
- PHP 8.x
- Composer
- Node.js y npm

## Backend (Symfony)
```bash
cd my-api
php -S localhost:8000 -t public
```

## Frontend (React)
```bash
cd my-frontend
npm install
npm start
```

Abre http://localhost:3000 en el navegador.

## Uso
1. Ingresa dos números en los campos `a` y `b`.
2. Haz clic en **Calculate**.
3. El resultado aparece debajo del botón.
